# Lembang Heritage Tour - Digital Companion

A professional Progressive Web App (PWA) designed to enhance the touring experience in Lembang. This app serves as a digital guide, providing historical narratives and geological insights for guests.

## ✨ Features
- **Offline Access:** Powered by Service Workers to ensure content is available even without an internet connection.
- **Heritage Narratives:** Detailed stories about Lembang's colonial secrets and the Ursone family legacy.
- **Geological Insights:** Information regarding the Lembang Fault and regional geology.
- **Responsive Design:** Optimized for mobile devices used on-the-go by tour guests.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **PWA Features:** Web App Manifest & Service Workers
- **Deployment:** GitHub Pages

## 🚀 How to Use
1. Visit the [Live Demo](URL_LINK_GITHUB_PAGES_ANDA_DISINI)
2. On your mobile browser, select "Add to Home Screen" to install it as an app.
3. Start exploring the heritage of Lembang!
